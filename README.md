# Projeto de Questões em C#

Este projeto contém a implementação de cinco questões diferentes em C#.

## Uso

Para executar o programa, siga os passos abaixo:

1. Compile o programa.
2. Execute o programa.
3. Escolha uma das questões (1 a 5) ao ser solicitado.

## Arquivo JSON

Certifique-se de que o arquivo `faturamento.json` esteja no mesmo diretório do executável. O conteúdo do arquivo deve ser:
